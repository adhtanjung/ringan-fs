Assessment Test

Open the browser console and navigate to http://localhost:3001/application/chatbot

Then send a message like: "I'm feeling very stressed and anxious about work. I can't sleep and feel overwhelmed."

This should trigger the assessment flow and you should see debug information in the console.

What to look for in console:

  • Debug logs showing assessment data structure
  • response_type field in the question object
  • Whether the 1-10 scale UI appears

Expected behavior:

  • After a few exchanges, the system should offer an assessment
  • When you accept, you should see assessment questions
  • For scale questions, a 1-10 rating interface should appear
